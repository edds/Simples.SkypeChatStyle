Simples a Skype Chat Style

by Edd Sowden

This currently won't work with the Skype for Mac 5.0 Beta. I will be updating it shortly though. If you wish to live on the edge a very early alpha version can be downloaded now.

Fed up with the default Skype chat style? I decided to make my own, you can use it too.

Installation Instructions

  1. Download the latest Simples.SkypeChatStyle release
  2. Double click Simples.SkypeChatStyle-0.3.zip to extract it
  3. Double click the extracted Simples.SkypeChatStyle bundle to install
  4. Select the Simples chat style in the Skype chat preferences

Screen Shot

Code

The code is heavily based on the 'Skype Modern Chat Style'. I have however, altered it to make use of the fact that the chat window is running a recent Webkit. As it has access to CSS3 features it seemed obvious to make use of them, such as animations, rounded corners and gradients.
